# Cheat sheet for keybin

## Tmux

New session

```
tmux new -s <session_name>
```

List all available sessions

```
C-a s
```

Split vertically

```
C-a |
```

Split horizontally

```
C-a -
```

Create new window

```
C-a c
```

Move to specific window

```
C-a <number>
```

Rename window

```
C-a ,
```

List available windows

```
C-a w
```

Install plugins

```
C-a I
```

**Movement**

Move to left Pane

```
C-a h
```

Move to down Pane

```
C-a j
```

Move to down Pane

```
C-a k
```

Move to right Pane

```
C-a l
```

**Copy mode**

Allows movement with vim specific keys

```
C-a [
```

Exit copy mode

```
C-a c
```
